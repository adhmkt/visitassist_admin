
# Markdown Cheat Sheet

Markdown is a lightweight markup language for formatting text. This guide covers the basic and advanced syntax of Markdown.

---

## **Basic Markdown Syntax**

### 1. Headers
Use `#` followed by a space to create headers.

```markdown
# Heading 1
## Heading 2
### Heading 3
#### Heading 4
##### Heading 5
###### Heading 6

2. Paragraphs
Leave a blank line between paragraphs.


This is the first paragraph.

This is the second paragraph.

3. Bold and Italics
Bold: Use ** or __.

Italic: Use * or _.

Bold and Italic: Use *** or ___.

This is **bold** text.
This is *italic* text.
This is ***bold and italic*** text.
4. Lists
Unordered Lists
Use -, *, or + followed by a space.

markdown
Copy
- Item 1
- Item 2
  - Subitem 2.1
  - Subitem 2.2
- Item 3
Ordered Lists
Use numbers followed by a period and a space.

markdown
Copy
1. First item
2. Second item
3. Third item
5. Links
Use [text](URL) to create a hyperlink.

markdown
Copy
[Google](https://www.google.com)
6. Images
Use ![alt text](image URL) to embed an image.

markdown
Copy
![Markdown Logo](https://markdown-here.com/img/icon256.png)
7. Code
Inline Code
Use backticks (`) for inline code.

markdown
Copy
This is `inline code`.
Code Blocks
Use triple backticks (```) for code blocks.

markdown
Copy
```python
def hello_world():
    print("Hello, World!")
```
8. Blockquotes
Use > to create a blockquote.

markdown
Copy
> This is a blockquote.
> It can span multiple lines.
9. Horizontal Rules
Use ---, ***, or ___ to create a horizontal line.

markdown
Copy
---
10. Tables
Use pipes (|) and dashes (-) to create tables.

markdown
Copy
| Column 1 | Column 2 | Column 3 |
|----------|----------|----------|
| Row 1    | Data 1   | Data 2   |
| Row 2    | Data 3   | Data 4   |
11. Strikethrough
Use ~~ to strikethrough text.

markdown
Copy
This is ~~strikethrough~~ text.
12. Emojis
Use emojis directly in Markdown.

Advanced Markdown Features
1. Footnotes
Use [^1] for footnotes.

markdown
Copy
Here is a footnote reference[^1].

[^1]: This is the footnote.
2. Task Lists
Use - [ ] for unchecked tasks and - [x] for checked tasks.

markdown
Copy
- [x] Write the introduction
- [ ] Add examples
- [ ] Review the document
3. Escaping Characters
Use a backslash (\) to escape special characters.
